## 🚀 Installation Guide

### 📦 Prerequisites
- ✅ JDK 17+
- ✅ Apache NetBeans 13
- ✅ Apache Tomcat 10

### 🛠 Setup Steps
1. **Configure JDK**:  
   `Tools → Java Platforms → Add JDK 17`

2. **Configure Tomcat**:  
   `Tools → Servers → Add Tomcat 10`

3. **Run Project**:  
   - Open project in NetBeans  
   - Right-click project → `Run`

| Role     | Capabilities                          |

| Guest    | Browse books                          |

| Admin    | Full system control                   |

## 🌟 Key Features

### 🛒 Shopping Features
- Book browsing with filters/sorting  
- Cart management  
- Order processing  
- Voucher system  
- Book feedback

### 🛠️ Admin Features
- Book management  
- Staff management  
- Inventory control  
- Sales analytics  
- Chatbot monitoring

### 📊 Statistics Module
- Revenue reports  
- Book performance  
- Order tracking  
- Data export (Excel)
